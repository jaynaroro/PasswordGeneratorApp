# Password Generator Android App

## Features
- Random secure password generation
- Local password vault
- PIN-protected access
- Room database persistence
- Copy to clipboard
- Show/hide passwords
- Cybersecurity-inspired UI

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- Material 3
